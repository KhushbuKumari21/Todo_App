# React To-Do List (Vite)

A simple and elegant To-Do List built using **React + Vite** .
- Add tasks
- Delete tasks
- Edit tasks (inline)
- Mark tasks as completed
- Uses functional components, state & props

**##src**
App.jsx – main component with state & logic

**##Components**
Header.jsx – title and subtitle
TodoList.jsx – displays all tasks
TodoItem.jsx – single task card

**##Features**
Add new To-Do
Delete To-Do
Mark complete
Edit To-Do
Styled UI

**## Run Program**
1.Install dependencies
npm install

2.Run development server
npm run dev

3 Open browser
http://localhost:5174/


**## Technologies Used**
- React
- Vite
- CSS3
